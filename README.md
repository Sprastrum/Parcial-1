# V. Conceptos
### 1. Encapsulamiento:
Encapsulamiento es mantener una información constante, 
así se evita que otras clases modifiquen la información y para mantener uso 
único a cada clase.


### 2. Ocultamiento
El ocultamiento es mantener la información escondida de otras clases.
